v3-blog 开发记录

主要功能

  • 博文列表展示
  • 分页展示
  • 搜索框实时查询
  • 博文分类切换查看
  • 后台登录
  • 文章、分类管理
  • 富文本编辑
  • 图片上传
  • 登录验证，token 技术

开发技术栈

npm 装包卡顿，更换源：
npm config set registry https://registry.npmmirror.com/
查看源：npm config get registry

  • 服务端：
    • Express 框架
    • multer 包（上传文件）
    • sqlite3（数据库）
    • uuid（生成安全 id）
  • 前端：Naive UI（TypeScript）

服务端开发

  1. 新建空项目，安装 express 包（Tips:可以直接 install）
  2. 安装 multer sqlite3 uuid
  3. 安装 json 支持app.use(express.json())
  4. 设置开放跨域请求
  5. 设置上传文件中间件 multer 即上传目录
  6. 配置数据库
    • 数据库软件：heidisql
    • 数据库选择：sqlite
  7. 测试路由
  8. 数据库查询优化，使用 promise
