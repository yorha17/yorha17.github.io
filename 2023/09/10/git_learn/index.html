git 学习记录

基础指令

git init

将文件夹初始化为一个 git 仓库

git status

查看所有文件状态

git add <filename>

将未跟踪状态的文件加入暂存区

git commit -m "提交注释内容"

将暂存区文件提交到仓库

git log

查看提交历史

git log 命令参数

git reset

版本回退

git branch

查看所有分支

git branch <name>

创建名为 name 的分支

通过git log命令查看创建分支效果

在当前分支（master 分支）提交文件后查看 log

git checkout <branch_name>

更换当前分支

能看到 HEAD 指向分支 dev01

可以直接切换到一个不存在的分支（创建并切换）

git checkout -b <branch_name>
